    Elixir v1.3.0-dev Calendar

    This module defines the responsibilities for working with calendars, dates, times and datetimes in Elixir.

    Currently it defines types but may define the calendar behaviour in future versions. For the actual date, time and datetime structures, see Date, Time, NaiveDateTime and DateTime.

    Note the year, month, day, etc designations are over specified (i.e. an integer instead of 1..12 for months) because different calendars may have a different number of numbers and so on.
